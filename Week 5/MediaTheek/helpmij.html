Data ophalen

Voor de Visual Studio liefhebbers

Wie gebruik maakt van Visual Studio en JSON uitleest op IIS, moet de volgende configuratie toevoegen in de web.config:

<!-- toevoegen voor specifieke file types, anders not found -->
<system.webServer>
    <staticContent>
        <mimeMap fileExtension=".json" mimeType="application/json" />
    </staticContent>
</system.webServer>

Lokaal data ophalen wanneer je chrome gebruikt

Bij lokaal werken met Chrome, veroorzaakt een webworker een security error. Wil je dat verhinderen, start dan Chrome op met volgende argumenten: chrome.exe - -allow-file-access-from-files.

Chart.js

De data

De data die je grafiek nodig heeft is een array van items die info geeft over elk element. In ons geval zal die er ongeveer zo uit zien:

[
    {
        value: 10,
        color: kevinderudder.color.randomColor(),
        highlight: "#FF9999",
        label: "multi-mania"
    },
    {
        value: 8,
        color: kevinderudder.color.randomColor(),
        highlight: "#FF9999",
        label: "nmct"
     },
]
